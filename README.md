# jesaispasquoidessiner
Générateur d'idées de trucs à dessiner.

Demo : [jesaispasquoidessiner.fr](http://jesaispasquoidessiner.fr)

Made with Bottle.

## HOWTO
Run
```python gen-db.py```
pour générer la base de données,
```python app.py```
pour lancer l'application.
